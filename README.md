# Sprint 1 – HTML & CSS Layout

Simple static page to practice flexbox, grid, and responsive media queries.

## Project structure

- `index.html` – semantic layout with header, banner, content, ads, and footer.
- `EstilosPrueba.css` – main styles (desktop/tablet/phone).

## Running the page

Open `index.html` directly in your browser or via Live Server. No build step needed.

## Customizing

- Adjust colors/spacing in `EstilosPrueba.css`.
- Tweak grid columns in `.articles` and wrapper padding (`header`, `main`, `.pie`) for different proportions.
